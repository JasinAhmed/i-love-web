Daily Checkout

  1. Wat is het verschil tussen Flexbox Layout en Grid Layout? Wanneer zou je welke gebruiken?
    Flexbox is bedoeld voor eenrichtings-layouts (horizontaal of verticaal) en is handig voor het uitlijnen van items in een rij of kolom. Grid is bedoeld voor tweedimensionale layouts, waarbij je zowel rijen als kolommen tegelijk beheert. Ik zou Flexbox gebruiken voor kleinere componenten of simpele alignments (zoals navigatiebalken), en Grid voor complete paginalayouts of complexe structuren.
  2. Wat is de default Layout Mode van elk element?
    Standaard hebben elementen een flow layout (block of inline). Block elementen nemen de hele breedte in en beginnen op een nieuwe regel. Inline elementen staan in de tekststroom en nemen alleen de ruimte van hun inhoud in.
  3. Schrijf op welke Layout Modes je per onderdeel nodig hebt voor het ontwerp van de opdrachtgever.
    Voor de grote structuur zou ik CSS Grid gebruiken (zoals de hoofdindeling van de pagina). Voor kleinere onderdelen, zoals knoppen, navigatie en formulieren, zou ik Flexbox gebruiken. Voor tekst en standaardinhoud gebruik ik de standaard flow layout.
